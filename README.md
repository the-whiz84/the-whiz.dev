# Portfolio - Kinetic Typography Edition

A high-impact portfolio featuring massive kinetic typography, fluid animations, and a bold modern aesthetic.

## Design Philosophy

- **Theme**: Kinetic Typography & Motion
- **Core Visuals**: Large scrolling text, mask animations, smooth marquee effects
- **Typography**: `Monument Extended` (bold headers) + `Inter` (body)
- **Colors**: Dark theme with orange accents

## Tech Stack

- **React + Vite** (Fast build & HMR)
- **Tailwind CSS v4** (Styling)
- **Framer Motion** (Complex text animations)
- **Lucide React** (Iconography)

## Getting Started

1. **Install dependencies:**
   ```bash
   bun install
   ```

2. **Run development server:**
   ```bash
   bun dev
   ```

3. **Build for production:**
   ```bash
   bun run build
   ```
