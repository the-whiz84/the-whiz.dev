# The-Whiz.dev (Isometric Edition)

This is the source code for my personal portfolio website, [the-whiz.dev](https://the-whiz.dev).
This version features a **3D Isometric Design**, glassmorphism UI, and fluid animations for a modern, immersive experience.

## üöÄ Getting Started

To run this project locally, follow these steps:

### Prerequisites

-   **Bun**: Latest version (Required)

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/the-whiz84/the-whiz.dev.git
    cd the-whiz.dev
    ```

2.  **Install dependencies**:
    ```bash
    bun install
    ```

3.  **Run the development server**:
    ```bash
    bun run dev
    ```

4.  **Open the browser**:
    Navigate to `http://localhost:5173` to see the result.
