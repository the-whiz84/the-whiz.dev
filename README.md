# Portfolio - Neon Synthwave Edition

A retro-futuristic portfolio featuring a cyberpunk neon aesthetic with grid animations and glitch effects.

## Design Philosophy

- **Theme**: Retro-Futuristic Synthwave
- **Core Visuals**: Neon grids, scanlines, vivid magenta/cyan palette, glitch text effects
- **Typography**: `Press Start 2P` (headings) + `Rajdhani` (tech/UI)

## Tech Stack

- **React + Vite** (Fast build & HMR)
- **Tailwind CSS v4** (Styling)
- **Framer Motion** (Animations)
- **Lucide React** (Iconography)

## Getting Started

1. **Install dependencies:**
   ```bash
   bun install
   ```

2. **Run development server:**
   ```bash
   bun dev
   ```

3. **Build for production:**
   ```bash
   bun run build
   ```
